About This Page

This page shows how different CSS files can change the look of the same HTML. By pressing the style buttons, you can instantly switch between multiple designs without changing the actual content.

HTML provides the structure of a webpage. CSS controls colors, fonts, spacing, and layout. With separate stylesheets, we can make one page appear in different themes.

This technique is useful for switching between light and dark modes, or for making special print-friendly versions of a site.

Try pressing Style 1, Style 2, Style 3, and No Style on the right to see the difference in design.

Even though the content remains the same, the presentation can change completely. This is the power of CSS in web development.
